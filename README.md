# Music Generation with AI

## Project Overview

This project is a simple Music Generation AI application developed using Python. The program generates random musical notes and creates a MIDI music file automatically.

## Features

* Generates random musical notes.
* Creates a MIDI (.mid) music file.
* Easy to understand and use.
* Beginner-friendly AI project.

## Technologies Used

* Python
* music21 Library
* Random Module

## Project Files

* `app.py` – Main Python program for music generation.
* `generated_music.mid` – Generated music output file.
* `requirements.txt` – Required libraries.
* `README.md` – Project documentation.

## How to Run the Project

1. Open the project folder in VS Code.

2. Install the required library:

   ```
   pip install music21
   ```

3. Run the program:

   ```
   python app.py
   ```

4. After successful execution, a file named:

   ```
   generated_music.mid
   ```

   will be generated automatically.

## Output

The program displays:

```
Music generated successfully!
```

and creates a MIDI music file named `generated_music.mid`.

## Conclusion

This project demonstrates a simple implementation of AI-based music generation using Python. It automatically generates musical notes and stores them as a MIDI file.
